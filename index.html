One continuous loop

From observation to tested change

curIAtor puts the feedback channel around the running app, then carries the app, thread, screenshot, source scope, and smoke command into the agent task.

  1. InspectOpen any app through one collection shell.
  2. CaptureRate, comment, screenshot, annotate, or reply in-thread.
  3. ActThe curIAtor skill opens the exact task bundle in Codex or Claude Code.
  4. VerifySmoke-test, reload, reply, and retain the Git and SQLite receipt.
